# Bank-Customer-Churn-Prediction-System

# 🏦 Bank Customer Churn Prediction Using Machine Learning

## 📌 Project Overview

Customer attrition poses a significant financial risk to retail banks, directly eroding profitability and diminishing long-term market share.This project leverages machine learning classification models to proactively detect early behavioral indicators of churn, enabling the institution to deploy targeted, data-driven retention interventions



The application is built using **Python**, **Scikit-learn**, and **Streamlit**, and is deployed on **Hugging Face Spaces** with an interactive web interface.

---

## 🎯 Problem Statement

Customer attrition poses a significant financial risk to retail banks, directly eroding profitability and diminishing long-term market share.

Traditional, reactive retention strategies are frequently inefficient, failing to identify at-risk accounts before the customer decides to terminate their relationship.

This project leverages machine learning classification models to proactively detect early behavioral indicators of churn, enabling the institution to deploy targeted, data-driven retention interventions
